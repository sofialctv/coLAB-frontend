# Vue CRUD Example

Esse é um template utilizando componentes genéricos para produzir CRUD's.

O exemplo está com Cidade!

As entidades Cliente e Pedido estão antigos.

## Executar o projeto

1 - Clone o repositório. <br>
2 - Instale os pacotes NPM.
``` sh
npm i
```
3 - Execute o projeto.
``` sh
npm run dev
```
